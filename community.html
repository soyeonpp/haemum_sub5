자주하는 질문

  • 계정 회원탈퇴 후 재가입을 할 수 있나요?
    네. 회원탈퇴를 하더라도 즉시 재가입이 가능합니다.
  • 계정회원정보는 어떻게 업데이트 할 수 있나요?
    화면 오른쪽 상단에
    회원정보>회원정보 수정 버튼을 클릭하여 업데이트할 수 있습니다.
  • 결제결제가 이중으로 되었어요. 어떻게 해야하나요?
    이중으로 결제가 된 경우 Q&A (1:1 고객문의, 고객센터 전화)를 이용하셔서 다음과 같은 사항을 알려주시면
    조취해드리겠습니다. (환불을 위해 카드번호-결제시간-결제금액을 미리 확인 후 이용해주시면 더욱 빠른 조취가 가능합니다.)
    오프라인 매장 이용시 해당 문제가 발생하시면 해당 매장 업주분께 결제 취소 요청을 하시면 더욱 신속하게 조취를 취해드립니다.
  • 점포점포 영업시간은 어디서 확인 할 수 있나요?
    홈페이지 상단의 STORE 창에서 해당 매장을 검색하시면 전화번호 및 영업시간을 확인하실 수 있습니다.
    공휴일의 경우 매장마다 상이하여 해당 매장에 전화하시어 확인 후 방문하시면 더욱 정확한 확인이 가능합니다.
  • 점포제 주변에 있는 해뭄 점포를 어떻게 찾을 수 있나요?
    홈페이지 상단의 STORE 창에서 위치정보를 동의하시면 점포선택 화면에 자동으로 현재위치 주변에 있는 점포가 나타납니다.
    안하셨다면 점포선택 화면에서 지역을 선택한 후 지도를 움직여 주변 점포를 확인하실 수 있습니다.
  • 계정회원가입을 하면 어떤 장점이 있나요?
    공지사항이나 이벤트 이용시 SNS 또는 메일을 통해 신속한 정보 획득이 가능하며,
    이용 회원 한정 쿠폰 사용이 가능합니다.
  • 계정주문내역은 어디서 확인 할 수 있나요?
    회원정보>주문내역 확인에 들어가시면 주문내역을 확인하실 수 있습니다.
  • 이용안내도시락 구입 후 몇 시간 안에 섭취해야 하나요?
    매장에서 조리 후 판매하기 때문에 구입 후 2시간이내 섭취를 권장하고 있습니다.
    또한 여름철에는 구입 후 바로 섭취하시는 것을 권장합니다.
  • 이용안내도시락 포장용기 전자레인지 사용 가능한가요?
    해뭄도시락 용기는 전자레인지 사용이 가능한 용기 입니다.
